## TRIGONOMETRY

	1. Trigonometric expand: sin(2x) -> 2sin(x)cos(x)
	2. Trigonometric reduce: 2sin(x)cos(x) -> sin(2x)
	3. Trigonometric to exponent: sinh(x) -> (e^x-e^(-x))/2
